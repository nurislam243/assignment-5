Question-2:What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText o textContent er moddhe parthokko holo innerHTML kono element er tag shoho text return kore. innerText kono element er tag sarai shudhu text return kore kintu hiden text return kore na. textContent hidden text shoho shob text return kore.

Question-3:What is event delegation in the DOM?

parent element e event listener jog kore child element er event guluke handle kora hoi jate kore dynamic vabe kono element jog korle notun kore oi element e event listener jog korte na hoi. ar ei poddhoti ke DOM er event delegation bole.

Question-4:What is event bubbling in the DOM?

kono child element e kono event ghotle shei event tar parent element er dike opore othe jai. ar etikei DOM e event bubbling bole. kono element e event ghotle tahole prothome shei event tar parent element er kase jai, tarpor tar grandparent , tarpor tar grandparent eivabe solte thake.
