Question-2:What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is_

  • innerHTML: innerHTML returns the text along with the tags of an element.
  • innerText: innerText returns only the text of an element without tags, but it does not return hidden text.
  • textContent: textContent returns all the text, including hidden text.

Question-3:What is event delegation in the DOM?

Adding an event listener to a parent element to handle child element events ensures that newly added elements dynamically receive the event without needing additional event listeners. This technique is called event delegation in the DOM.

Question-4:What is event bubbling in the DOM?

When an event occurs on a child element, it moves up to its parent element. This is called event bubbling in the DOM. When an event happens on an element, it first goes to its parent, then to its grandparent, and continues moving up the DOM hierarchy.

Question-5:How do you create, add, and remove elements using JavaScript?

Using JavaScript, elements can be created, added, and removed dynamically. The methods for creating, adding, and removing elements are given below:

  • Create: To create an element using JavaScript, the document.createElement() method is used.
  • Add: To add an element, the appendChild() method is used.
  • Remove: To remove an element, the removeChild() method is used.
